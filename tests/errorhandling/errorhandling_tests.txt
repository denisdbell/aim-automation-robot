#US749 - Error handling (404 & 500)
I should see a 404 when I navigate to an invalid URL
	[Tags]		ErrorHandling	404
	Given I am a MyBlue Member on the Login Page
	When I navigate to "${SERVER}/invalidURL"
	Then I should see the element ".errorPage" displayed
	
I should see a message informing me of the error
	[Tags]		ErrorHandling	404
	Then I should see ".message" have the text "The web address you are trying to access is not a functioning page on our site." displayed
	
I should see message code on the 404 page
	[Tags]		ErrorHandling	404
	Then I should see the element ".messageCode" displayed
	
I should see a messageCode informing me of the error
	[Tags]		ErrorHandling	404
	Then I should see ".messageCode" have the text "Error 404: Page not found" displayed
	Teardown Browser
